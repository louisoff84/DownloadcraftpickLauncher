Téléchargez le Craftpick Launcher

Version : 1.0.5

Sélectionnez votre système d'exploitation ci-dessous pour télécharger la version optimisée du **Craftpick Launcher**. Compatible avec Windows, Linux et Mac.

TÉLÉCHARGER Windows (64-bit) .exe TÉLÉCHARGER Linux (AppImage) TÉLÉCHARGER macOS (Universal .dmg)

Instructions d'installation :

Windows (.exe) : Double-cliquez sur le fichier `.exe` téléchargé et suivez les instructions.
macOS (.dmg) : Ouvrez le fichier `.dmg` et faites glisser l'icône de l'application **Craftpick Launcher** dans votre dossier `Applications`.
Linux (AppImage) :
  1. Téléchargez le fichier `.AppImage`.
  2. Rendez-le exécutable : ouvrez votre terminal, naviguez vers le dossier de téléchargement et tapez : chmod +x craftpick-Launcher-linux-x86_64.AppImage
  3. Exécutez-le : double-cliquez ou tapez : ./craftpick-Launcher-linux-x86_64.AppImage
